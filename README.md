# Responsive Webpage using CSS Media Queries

This webpage was created and made mobile-friendly using **VS Code**.

## What Was Done
- Designed the webpage using **HTML** and **CSS** in VS Code.  
- Added **media queries** to make the layout responsive for mobile screens.  
- Adjusted images, text size, and layout for smaller devices.  
- Tested the design using **Chrome DevTools** inside VS Code Live Server.

## Files
- `index.html` – main webpage  
- `style.css` – contains styles and responsive media queries
